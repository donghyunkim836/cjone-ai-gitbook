# 외부 도구 연결(Plugin)

## ⏵ 플러그인(Plugin) 활용 가이드

> 스마트폰 앱처럼, Claude에 필요한 기능을 골라서 추가하는 방법

***

### 플러그인이란?

Claude의 기본 기능으로 안 되는 일이 있을 때, **플러그인을 설치하면 그 기능이 생깁니다.**

스마트폰에 처음부터 모든 앱이 깔려 있지 않듯, Claude도 기본 상태에선 일부 기능이 없습니다. 하지만 플러그인 마켓플레이스에서 필요한 것을 골라 설치하면 바로 사용할 수 있습니다.

> **비유하자면** — 스마트폰에서 카카오뱅크를 쓰려면 앱을 설치해야 하듯, Claude에서 Word 문서를 만들려면 docx 플러그인을 설치하면 됩니다.

플러그인 하나를 설치하면 관련 **스킬(작업 명령어)** 과 외부 서비스 연결 기능이 함께 추가됩니다.

***

### 플러그인 설치하기 (데스크탑 앱 기준)

#### Step 1. 플러그인 패널 열기

데스크탑 앱 왼쪽 사이드바에서 플러그인 아이콘을 클릭합니다. 또는 채팅창에 아래처럼 입력해도 됩니다.

```
어떤 플러그인을 사용할 수 있어?
```

#### Step 2. 원하는 플러그인 찾기

마켓플레이스에서 이름이나 기능으로 검색합니다.

**자주 쓰는 플러그인 예시:**

| 플러그인                | 추가되는 기능       | 이런 상황에        |
| ------------------- | ------------- | ------------- |
| **docx**            | Word 문서 생성·편집 | 기획서, 보고서, 제안서 |
| **pptx**            | 파워포인트 슬라이드 생성 | 발표 자료 초안      |
| **xlsx**            | 엑셀 스프레드시트 생성  | 데이터 정리, 예산표   |
| **pdf**             | PDF 내용 추출·변환  | PDF 분석, 재편집   |
| **service-ia**      | 서비스 정보구조 문서화  | 서비스 기획        |
| **service-process** | 업무 프로세스 흐름도   | 업무 절차 시각화     |

#### Step 3. 설치 및 바로 사용

플러그인을 선택하고 **설치** 버튼을 누르면 즉시 사용 가능합니다.

***

### 플러그인 설치 후 사용 방법

#### 자연어로 요청

설치 후에는 평소처럼 말하면 됩니다. Claude가 알아서 플러그인을 활용합니다.

```
이 내용을 파워포인트로 만들어줘
→ pptx 플러그인이 자동으로 작동
```

```
지난달 판매 데이터를 엑셀로 정리해줘
→ xlsx 플러그인이 자동으로 작동
```

#### 슬래시 명령어로 직접 호출

```
/pptx

2025년 하반기 마케팅 전략 발표자료를 만들어줘.
주요 내용:
- 목표: 브랜드 인지도 30% 향상
- 전략: SNS 집중, 인플루언서 협업
- 예산: 5억 원
- 기대효과: 신규 고객 유입 50% 증가
슬라이드 10장, 깔끔하고 전문적인 디자인으로
```

***

### 실전 활용 예시

#### 상황 1 — 회의 직후 즉시 보고서 완성

```
/docx

오늘 파트너사 미팅 내용을 보고서로 만들어줘.

일시: 2025년 6월 3일 오후 2시
참석자: 우리 측 3명, 파트너사 2명
주요 논의:
- 공동 마케팅 캠페인 9월 론칭 확정
- 예산 분담: 우리 60%, 파트너 40%
- 다음 미팅: 6월 17일

형식: A4 2장 이내, 임원 보고용
```

결과물을 바로 다운로드해서 공유할 수 있습니다.

#### 상황 2 — 데이터를 엑셀 표로 정리

```
/xlsx

아래 데이터를 팀별 월간 실적 현황표로 만들어줘.
헤더: 팀명, 목표, 실적, 달성률, 전월 대비
영업1팀: 목표 10억, 실적 8.7억
영업2팀: 목표 8억, 실적 9.2억
마케팅팀: 목표 5억, 실적 4.1억

달성률과 전월 대비는 자동 계산 수식으로 넣어줘
```

#### 상황 3 — PDF에서 필요한 내용만 추출

```
/pdf

첨부한 계약서 PDF에서
- 계약 기간
- 금액 및 지급 조건
- 해지 조건
만 뽑아서 표로 정리해줘
```

***

### 플러그인 관리

설치된 플러그인은 사이드바 플러그인 패널의 **설치됨(Installed)** 탭에서 확인하고 관리할 수 있습니다.

* **비활성화**: 일시적으로 꺼두고 싶을 때
* **삭제**: 더 이상 필요 없을 때
* **업데이트**: 새 버전이 나왔을 때 알림이 표시됩니다

***

### 알아두면 좋은 것

* 플러그인은 외부 개발자가 제작한 것도 있습니다. **공식 마켓플레이스에 등록된 것**만 설치하세요.
* 설치 시 어떤 권한을 요청하는지 확인하는 것이 좋습니다.
* 같은 기능의 플러그인이 여러 개 설치되어 있으면 충돌이 생길 수 있습니다. 하나만 유지하세요.
* 사내 전용 플러그인이 있다면 AI 퍼실리테이터에게 문의하세요.
