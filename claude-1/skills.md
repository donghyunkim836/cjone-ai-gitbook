# 반복 작업 설정(Skills)

## ⏵ 스킬(Skills) 활용 가이드

> Claude에게 "이 작업은 항상 이 방식으로 해줘"라고 미리 약속해두는 기능

***

### 스킬이란?

스킬은 **자주 반복하는 작업을 하나의 명령어로 실행**할 수 있게 해주는 기능입니다.

매번 길게 설명하지 않아도 됩니다. 예를 들어 "회의록 정리" 스킬을 만들어두면, 녹취록을 붙여넣고 `/meeting-notes`라고 입력하는 것만으로 날짜·참석자·안건 추출, 논의 요약, 액션아이템 표 정리까지 한 번에 완료됩니다.

> **비유하자면** — 자주 시키는 업무를 팀원에게 한 번 제대로 알려줬더니, 다음부터는 "저번처럼 해줘"라는 말 한 마디로 끝나는 것과 같습니다.

***

### 스킬 사용 방법

#### 1. 슬래시(/) 명령어로 호출

채팅창에 `/`를 입력하면 사용 가능한 스킬 목록이 나타납니다. 원하는 스킬을 선택하거나 직접 입력합니다.

**예시 — 회의록 정리**

```
/meeting-notes

[녹취록 내용을 여기에 붙여넣기]
```

**예시 — Word 문서 생성**

```
/docx

3분기 영업 실적 보고서를 만들어줘. 주요 내용은 다음과 같아:
- 매출 목표 대비 달성률 87%
- 신규 고객 32건 확보
- 다음 분기 전략: 중소기업 타겟 집중
```

#### 2. 자연어로 요청하면 Claude가 자동으로 스킬 활용

직접 호출하지 않아도 됩니다. 관련된 작업을 요청하면 Claude가 적합한 스킬을 자동으로 판단해 실행합니다.

```
이 데이터를 엑셀 파일로 정리해줘
→ Claude가 자동으로 xlsx 스킬 활용
```

***

### 현재 사용 가능한 스킬 예시

데스크탑 앱에 설치된 플러그인에 따라 다르지만, 대표적인 스킬은 다음과 같습니다.

| 스킬                 | 하는 일            | 이런 상황에 써요        |
| ------------------ | --------------- | ---------------- |
| `/docx`            | Word 문서 생성 및 편집 | 보고서, 기획서, 제안서 작성 |
| `/pptx`            | 파워포인트 슬라이드 생성   | 발표 자료 초안 만들기     |
| `/xlsx`            | 엑셀 스프레드시트 생성    | 데이터 정리, 표 만들기    |
| `/pdf`             | PDF 처리 및 생성     | PDF에서 내용 추출, 변환  |
| `/service-ia`      | 서비스 정보구조(IA) 문서 | 서비스 기획 문서화       |
| `/service-process` | 업무 흐름도(플로우차트)   | 프로세스 시각화         |

***

### 나만의 스킬 만들기

반복하는 작업이 있다면 Claude에게 직접 만들어달라고 요청하면 됩니다. 파일을 직접 편집할 필요가 없습니다.

#### 요청 방법

Claude에게 이렇게 말하면 됩니다:

```
나만의 스킬을 만들어줘.
스킬 이름: weekly-report
설명: 주간 업무 보고서를 작성해주는 스킬
내용:
- 이번 주 완료한 업무 목록을 받으면
- 업무별 진행률과 결과를 요약하고
- 다음 주 계획을 표로 정리하고
- 전달 사항이 있으면 별도 섹션으로 작성해줘
```

#### 만들어진 스킬 사용

```
/weekly-report

이번 주 완료: 신규 거래처 미팅 3건, 제안서 2건 발송, 계약 1건 완료
다음 주 계획: 후속 미팅 일정 잡기, 신규 리드 발굴
전달 사항: 다음 달 영업 목표 상향 조정 예정
```

***

### 실전 활용 예시

#### 상황 1 — 매주 반복하는 보고서

매주 같은 형식으로 보고서를 작성해야 한다면, 형식을 스킬로 등록해두세요.

```
경영진 보고용 스킬을 만들어줘.
스킬 이름: exec-summary
내용: 전달받은 내용을 경영진 보고 형식으로 정리해줘.
- 핵심 요약 3줄
- 현황 및 주요 수치
- 리스크 및 대응 방안
- 다음 액션 아이템 (담당자, 기한 포함)
한 페이지를 넘기지 말 것
```

이후 매주:

```
/exec-summary

[이번 주 내용 붙여넣기]
```

#### 상황 2 — 외부 발송 이메일

거래처나 고객에게 보내는 이메일의 톤앤매너를 스킬로 고정해두면 일관성이 유지됩니다.

```
외부 공식 이메일 스킬을 만들어줘.
스킬 이름: biz-email
내용: 전달받은 상황을 바탕으로 비즈니스 이메일 초안을 작성해줘.
- 제목 포함
- 격식체 사용 (합니다체)
- 인사말 → 본문 → 마무리 인사 → 서명란 구조
- 500자 이내로 간결하게
```

```
/biz-email

상황: 납기 지연 안내. 원자재 수급 문제로 납기가 2주 지연됩니다.
수신: 거래처 구매담당자
```

***

### 팀에서 스킬 공유하기

한 명이 만든 스킬을 팀 전체가 쓸 수 있습니다. 스킬 파일을 팀 공유 폴더나 사내 저장소에 올려두면, 팀원들이 동일한 스킬을 설치해서 사용할 수 있습니다. 구체적인 공유 방법은 담당자(AI 퍼실리테이터)에게 문의하세요.

***

### 알아두면 좋은 것

* 스킬이 많아질수록 `/`를 눌렀을 때 목록도 늘어납니다. 자주 쓰는 것 위주로 관리하세요.
* 스킬 설명(description)은 구체적으로 쓸수록 Claude가 적합한 상황에 자동으로 활용합니다.
* 스킬을 수정하면 **다음 대화부터 바로 반영**됩니다. 재시작이 필요 없습니다.
