# 👀 공간 소개

Claude를 효과적으로 활용하기 위한 페이지입니다. \
설치부터 실전 활용까지, 필요한 내용을 찾아 바로 쓸 수 있도록 정리하고있습니다.

***

## 🤖 Claude 시작하기

Claude Code를 처음 쓸 때 읽어야 할 내용입니다.

| 챕터            | 내용                         |
| ------------- | -------------------------- |
| 1. 사용 준비하기    | Desktop 앱과 CLI 설치, 첫 대화 시작 |
| 2. 대화 방식 이해하기 | 세션 관리, 파일 참조, 되돌리기         |
| 3. 응답 모델 설정하기 | 모델 선택, 사고 깊이 조절            |
| 4. 실행 권한 설정하기 | 권한 모드, Plan Mode, 민감 파일 보호 |
| 5. 대화 맥락 관리하기 | /clear, /compact, 맥락 용량 관리 |
| 6. 기존 규칙 유지하기 | 세션 간 규칙 유지, Auto Memory    |
| 7. 바로 써보기     | 다운로드 폴더 정리 실습              |

## 💡 Claude 활용하기

Claude Code를 효율적으로 활용하고자 할 때 읽어야 할 내용입니다.

<table><thead><tr><th width="359">챕터</th><th>내용</th></tr></thead><tbody><tr><td>반복 작업 설정(Skills)</td><td>스킬 설명, 사용법, 나만의 스킬 만들기</td></tr><tr><td>외부 도구 연결(Plugin)</td><td>플러그인 설명, 설치방법, 사용방법, 관리방법</td></tr><tr><td>외부 서비스 연결(MCP)</td><td>연결 방법, 활용 방법</td></tr></tbody></table>

## 📁 활용 사례

각 직무별 Skills/Plugin/MCP 활용 사례를 할 때 읽어야 할 내용입니다.
