# 📌 Ground Rule

생성형 AI의 올바른 사용은 업무 효율 향상과 보안 사고 예방의 첫걸음입니다. 모두가 안전하게 활용할 수 있도록, 생성형 AI 사용 시 유의사항을 안내드립니다.

***

### 🛡️ 정보보호 준수사항

* 서비스와의 대화 시 데이터를 학습 할 수 없도록 데이터 제어
* 개인정보, 민감정보, 영업비밀 입력 금지
* 회사의 보안관련 질문 발생 시 답변 자제

***

### 🖥️ 생성된 정보 활용 주의사항

* 교차 검증으로 제공된 답변의 정확성 확인
* 답변의 최신성/유효성 검증 필요
* 저작권 침해 방지를 위한 점검 필요
* 상업적 활용 시 법무/Compliance 부서 검토 필요

***

### 📋 구성원 Ground Rule

#### 1. 회사가 승인한 AI 도구 위주로 사용한다

회사가 공식 도입한 AI 도구를 업무에 우선 활용해보고, 새로운 도구 도입이 필요한 경우 담당 부서에 먼저 문의합니다.

#### 2. 최종 책임은 사용자 본인에게 있다

AI가 생성한 결과물이라도 이를 검토하고 활용하는 책임은 사용자에게 있습니다. "AI가 그렇게 했다"는 면책 사유가 되지 않습니다.

#### 3. 회사 대외비·기밀 자료는 입력하지 않는다

사내 등급 기준으로 '대외비' 이상으로 분류된 자료, 미발표 재무 정보, 계약 내용, 파트너사 정보 등은 AI에 입력하지 않습니다. 어느 등급인지 불명확한 자료도 입력 전 담당자에게 확인합니다.

#### 4. 생성된 결과물은 반드시 사람이 검토 후 사용한다

AI의 답변은 그대로 복사·붙여넣기 하지 않습니다. 사실 여부, 맥락 적합성, 표현의 적절성을 직접 확인한 후 활용합니다. 특히 수치·날짜·고유명사는 반드시 원본 자료와 대조합니다.

#### 5. 잘 된 활용 사례는 팀과 나눈다

효율이 크게 개선된 프롬프트나 활용 방식은 팀원과 공유합니다. 개인의 노하우가 팀 전체의 역량이 됩니다.

***

### ⚙️ 생성형AI 사용 시 아래와 같이 설정

#### 데이터 학습에 의한 외부 유출 방지

생성형 AI 서비스는 기본 상태에서 대화 내용을 모델 개선에 활용할 수 있습니다. 아래 절차에 따라 **데이터 학습 옵션을 반드시 꺼두세요.**&#x20;

#### Claude (웹/데스크탑)

> **참고**: Claude for Work(기업용 플랜) 사용자는 기본적으로 대화 내용이 학습에 사용되지 않습니다. 아래는 **개인 계정(Free·Pro·Max)** 사용자 대상 설정입니다.

1. **1단계**: [https://claude.ai](https://claude.ai) 접속 → 왼쪽 하단 또는 상단의 이름(계정) 클릭 → 드롭다운에서 **Settings** 선택
2. **2단계**: 설정창 왼쪽 메뉴에서 **Privacy** 클릭
3. **3단계**: **Help Improve Claude** 항목 확인 → 토글이 켜져 있으면 클릭하여 **꺼짐** 상태로 전환
4. **4단계**: 설정이 자동 저장됨 — 토글이 꺼진 상태로 표시되면 완료

> ⚠️ 설정을 끄면 **이후 새로운 대화**부터 학습에 사용되지 않습니다. 이미 진행 중인 학습 데이터는 소급 적용되지 않으므로, 최초 계정 생성 직후 설정하는 것을 권장합니다.

***

#### Claude (모바일 앱)

1. **1단계**: Claude 앱 실행 → 설정 메뉴에서 이름(계정) 선택
2. **2단계**: **Settings** 선택
3. **3단계**: **Privacy** 선택
4. **4단계**: **Help Improve Claude** 토글을 **꺼짐** 상태로 전환
