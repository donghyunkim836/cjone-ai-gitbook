# 07. 첫 실전 - 다운로드 폴더 정리

기본기를 배웠으니, 직접 해봅니다. 누구나 다운로드 폴더에 파일이 쌓여 있을 것입니다. CLI든 Desktop 앱이든 편한 쪽으로 Claude를 열고, 이 폴더를 정리해봅니다.

## Step 1. 다운로드 폴더에서 Claude 실행

CLI라면 다운로드 폴더로 이동해서 `claude`를 실행합니다. Desktop 앱이라면 Code 탭에서 다운로드 폴더를 선택합니다.

```bash
cd ~/Downloads
claude
```

처음 실행하는 폴더에서는 "이 폴더를 신뢰하느냐"는 확인이 나타납니다. 본인의 다운로드 폴더이므로 "Yes, I trust this folder"를 선택합니다.

실제 다운로드 폴더 대신 연습용으로 시작하고 싶다면:

```bash
mkdir ~/cleanup-practice
cd ~/cleanup-practice
claude
```

```
연습용으로 뒤섞인 파일들을 만들어줘.
보고서, 이미지, 스프레드시트, 프레젠테이션 등 여러 종류로 10~15개 정도.
```

## Step 2. 먼저 분석만 시켜보기

바로 정리를 시키지 않습니다. `Shift+Tab`으로 Plan 모드로 전환하고, 분석부터 요청합니다.

```
이 폴더의 파일들을 분석해서 정리 계획을 세워줘.
```

Claude가 파일 목록을 읽고 종류별로 분류한 계획을 보여줍니다. Plan 모드이므로 파일은 건드리지 않습니다. 계획이 마음에 들면 다음 단계로 넘어갑니다.

## Step 3. 정리 실행

`Shift+Tab`으로 Normal 모드로 돌아온 뒤, 원하는 방식을 구체적으로 지시합니다.

```
이 폴더의 파일들을 종류별로 정리해줘.
- 이미지 → images/
- 문서 → documents/
- 스프레드시트 → spreadsheets/
- 프레젠테이션 → presentations/
```

Claude가 하위 폴더를 만들고 파일을 이동합니다. 각 명령 실행 전 확인을 요청하므로, 검토 후 승인합니다.

## Step 4. 결과 확인

```
정리 결과를 트리로 보여줘.
```

## Step 5. 마음에 안 들면 되돌리기

파일 이동(`mv`)은 bash 명령어로 실행되므로, 체크포인트(`Esc Esc`)로는 되돌릴 수 없습니다. 대신 Claude에게 직접 요청합니다.

```
방금 정리한 거 원래대로 되돌려줘
```

이것은 체크포인트가 아니라 Claude의 대화 기억에 의존하는 방법입니다. 같은 세션 안에서만 가능하고, 세션을 종료하거나 컨텍스트가 압축되면 되돌릴 수 없습니다. 그래서 Step 2에서 Plan 모드로 먼저 확인하는 것이 중요합니다.

## 더 해보기

다운로드 폴더 외에도 다양한 정리를 시도해봅니다. Claude는 이미지를 직접 볼 수 있어서 사진 내용 기반 분류도 가능합니다.

```
이 폴더의 사진들을 보고 음식, 여행, 스크린샷, 문서 촬영으로 분류해줘

IMG_로 시작하는 사진들을 보고 내용에 맞는 파일명으로 바꿔줘. 예: 한강-야경.jpg

이 폴더에서 용량 큰 파일 10개를 찾아서 각각 뭔지 설명해줘

이 폴더의 영수증 사진들을 보고, 날짜_가게명_금액 형식으로 파일명을 바꿔줘

1년 넘게 안 열어본 파일들을 찾아서 목록으로 보여줘
```

> 사진이 많으면 토큰을 많이 소비합니다. 수십 장 단위로 나누어 작업하는 것을 권장합니다. 실제 중요한 폴더를 정리할 때는 Plan 모드로 먼저 확인하고, "삭제해줘"보다 "목록만 보여줘"를 먼저 사용합니다.
