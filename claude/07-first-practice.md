# 07. 첫 실전

지금까지 배운 걸 직접 써볼 차례입니다. 다운로드 폴더는 누구에게나 파일이 쌓여 있기 마련이라 연습하기 딱 좋습니다. CLI든 Desktop 앱이든 편한 걸로 열어서 정리해봅니다.

## 1. 다운로드 폴더에서 Claude 실행

CLI라면 다운로드 폴더로 이동해서 `claude`를 실행합니다. Desktop 앱이라면 Code 탭에서 다운로드 폴더를 선택합니다.

```bash
cd ~/Downloads
claude
```

처음 실행하는 폴더에서는 신뢰 여부를 묻는 화면이 나타납니다. 내 폴더이므로 "Yes, I trust this folder"를 선택합니다.

실제 폴더 대신 연습용으로 해보고 싶다면:

```bash
mkdir ~/cleanup-practice
cd ~/cleanup-practice
claude
```

```
연습용으로 뒤섞인 파일들을 만들어줘.
보고서, 이미지, 스프레드시트, 프레젠테이션 등 여러 종류로 10~15개 정도.
```

## 2. 분석 먼저

바로 정리하지 말고 `Shift+Tab`으로 Plan 모드로 전환한 뒤 분석부터 시킵니다.

```
이 폴더의 파일들을 분석해서 정리 방법을 알려줘.
```

Claude가 파일 목록을 읽고 분류 계획을 보여줍니다. Plan 모드라 파일은 건드리지 않습니다. 계획이 괜찮으면 다음으로 넘어갑니다.

## 3. 정리 실행

`Shift+Tab`으로 Normal 모드로 돌아와서 구체적으로 지시합니다.

```
이 폴더의 파일들을 종류별로 정리해줘.
- 이미지 → images/
- 문서 → documents/
- 스프레드시트 → spreadsheets/
- 프레젠테이션 → presentations/
```

명령 실행 전마다 확인을 요청하니 검토 후 승인합니다.

## 4. 결과 확인

```
정리 결과를 트리로 보여줘.
```

## 5. 되돌리기

파일 이동(`mv`)은 터미널 명령으로 실행되기 때문에 체크포인트(`Esc Esc`)로 되돌릴 수 없습니다. 같은 세션 안이라면 Claude에게 직접 부탁하면 됩니다.

```
방금 정리한 거 원래대로 돌려줘
```

세션을 종료하거나 맥락이 압축된 뒤에는 되돌리기 어렵습니다. 그래서 2단계에서 Plan 모드로 먼저 확인하는 게 중요합니다.

## 더 해보기

Claude는 이미지를 직접 볼 수 있어서 내용 기반 분류도 됩니다.

```
이 폴더의 사진들을 보고 음식, 여행, 스크린샷, 문서 촬영으로 나눠줘

IMG_로 시작하는 사진들을 보고 내용에 맞는 이름으로 바꿔줘. 예: 한강-야경.jpg

용량 큰 파일 10개를 찾아서 각각 뭔지 설명해줘

영수증 사진들을 보고 날짜_가게명_금액 형식으로 파일명 바꿔줘

1년 넘게 안 열어본 파일 목록 보여줘
```

> 사진이 많으면 토큰이 많이 듭니다. 수십 장 단위로 나눠서 작업하세요. 중요한 폴더를 정리할 때는 Plan 모드로 먼저 확인하고, "삭제해줘"보다 "목록만 보여줘"부터 시작하는 게 안전합니다.
